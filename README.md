# 🛒 Loja Homem de Ferro - Desafio DOM

Este projeto é uma landing page de uma loja temática desenvolvida para praticar a manipulação do DOM e a criação de interfaces responsivas.

## 🚀 Funcionalidades
* **Carrinho de Compras:** Adição e remoção dinâmica de produtos utilizando JavaScript.
* **Cálculos em Tempo Real:** Atualização automática do valor total ao adicionar produtos ou alterar quantidades.
* **Responsividade:** Layout adaptável para dispositivos móveis usando Media Queries.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica da loja e tabelas de carrinho.
* **CSS3:** Estilização personalizada com variáveis e foco em UX.
* **JavaScript (ES6+):** Lógica para manipulação de eventos e cálculos dinâmicos.

## 📂 Como rodar o projeto
1. Clone este repositório.
2. Abra o arquivo `index.html` no seu navegador.

## 📚 Créditos e Aprendizado
Este projeto foi desenvolvido acompanhando uma videoaula de JavaScript, servindo como base para o estudo de manipulação de listas e cálculos dinâmicos no navegador. 

A partir da aula, pude praticar:
* A lógica de captura de eventos em múltiplos botões.
* O uso de `replace` e `toFixed` para tratar valores monetários no Brasil.
* A estruturação de tabelas dinâmicas com `createElement`.
